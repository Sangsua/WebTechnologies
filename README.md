# Book Road
##### Book Road is a online marketplace for books.
##### This project is build for the course Modern Web Applications.

### Backend
- Python
- Flask
- SQLAlchemy, SQLite

### Frontend
- Bootstrap
- React 
- Jinja  

### How to use

1. clone repo
2. install pipenv
3. cd docs
4. pipenv shell
5. python app.py
6. install dependencies if necessary
