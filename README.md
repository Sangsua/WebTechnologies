# Book Road
##### Book Road is a online marketplace for books.
##### This project is build for the course Modern Web Applications.

### Backend
- Python
- Flask
- Jinja2 (Server side rending)

### Frontend
- Bootstrap
- React (is planned)

### How to use

1. Clone repo
2. install pipenv
3. pipenv shell
4. cd docs
5. python app.py
6. install dependencies if necessary
